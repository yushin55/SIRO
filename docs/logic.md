1. 경영학과 기준 직무 설명(봇이 안내할 내용)
1) 전략/경영기획

무슨 일을 하나요?
회사 전체의 방향을 정하고, “어디에 힘을 쓸지”를 결정합니다. 시장·경쟁사 분석, 매출/비용 구조 분석, 새로운 사업 기획, 연간 사업계획 수립 같은 일을 해요.

하루의 모습
리서치 → 데이터/자료 정리 → 파워포인트로 전략안 만들기 → 임원/팀에 설명하고 피드백 받기.

이런 분께 잘 맞아요

큰 그림 그리는 걸 좋아함

데이터·자료 근거로 말하는 걸 선호

발표·설득을 부담스럽지만 “하면 뿌듯한” 타입

2) 마케팅/브랜드

무슨 일을 하나요?
제품·서비스를 “어떻게 알릴지, 어떤 이미지로 기억되게 할지”를 고민합니다. 캠페인 기획, 광고·SNS 콘텐츠, 프로모션, 고객 분석 등을 해요.

하루의 모습
타깃 고객 정의 → 콘텐츠·이벤트 아이디어 회의 → 실행(카피·이미지 제작 협업) → 캠페인 결과(조회수·전환율 등) 분석.

이런 분께 잘 맞아요

사람들의 마음·반응에 관심 많음

글쓰기·콘텐츠·트렌드에 관심 많음

“재미있게 알리는 방법” 생각하는 걸 좋아함

3) 재무/회계

무슨 일을 하나요?
회사의 돈 흐름을 관리하고, 숫자로 회사의 건강 상태를 진단합니다. 재무제표 작성, 예산 수립·관리, 투자·자금 조달 검토 등을 합니다.

하루의 모습
매출·비용 데이터 정리 → 재무제표/리포트 작성 → 예산 대비 실적 분석 → 비용 절감·수익 개선 포인트 찾기.

이런 분께 잘 맞아요

엑셀·숫자·정확한 계산을 크게 거부감 없이 받아들임

꼼꼼하고 실수 줄이는 것에 강점

안정적이고 체계적인 업무 환경을 선호

4) 인사/HR(채용·조직문화·교육)

무슨 일을 하나요?
“사람”을 뽑고, 성장시키고, 오래 다니게 만드는 역할입니다. 채용, 조직문화, 평가·보상, 교육·러닝 등.

하루의 모습
채용 공고 기획 → 면접 진행 → 구성원 설문 분석 → 교육 프로그램 기획/운영 → 조직 문화 개선 프로젝트.

이런 분께 잘 맞아요

사람 이야기에 관심 많고, 상담·코칭에 흥미

갈등 조정, 분위기 메이킹을 자주 맡는 편

숫자도 보지만 “사람의 감정·동기”에 더 끌림

5) 운영/SCM/프로세스(물류·운영관리)

무슨 일을 하나요?
회사의 상품이나 서비스가 “제때, 적절한 비용으로” 잘 움직이게 만드는 직무입니다. 재고 관리, 물류·배송, 매장 운영, 내부 프로세스 개선 등을 합니다.

하루의 모습
물류/재고 현황 확인 → 병목 구간 찾기 → 프로세스 개선안 설계 → 현장과 협업해서 실험 → 효율성 지표(리드타임, 비용 등) 체크.

이런 분께 잘 맞아요

현실적인 문제를 눈으로 보고 개선하는 걸 좋아함

체계·표준·매뉴얼 만드는 것에 흥미

“어떻게 하면 더 빠르고 싸게 할까?”를 자주 생각함

6) 영업/B2B 세일즈·사업개발(BD)

무슨 일을 하나요?
회사의 제품/서비스를 실제 고객에게 팔고, 파트너십·거래처를 만드는 역할입니다. 매출을 직접적으로 만드는 최전선 직무예요.

하루의 모습
잠재 고객 리스트업 → 미팅·전화·메일로 제안 → 견적/조건 협상 → 계약 체결 → 이후 관계 관리.

이런 분께 잘 맞아요

사람 만나서 이야기하는 걸 비교적 즐기는 편

목표(매출, 계약 수) 달성에 대한 성취감이 큼

거절을 당해도 다시 도전할 수 있는 멘탈

7) 데이터/비즈니스 애널리틱스(경영+데이터)

무슨 일을 하나요?
매출·고객·마케팅 성과 데이터를 분석해서, “어디에 더 투자해야 할지”를 제안하는 역할입니다. 마케팅/전략과 데이터 사이에 서 있는 포지션.

하루의 모습
데이터 추출(SQL/엑셀) → 시각화(대시보드) → 패턴·인사이트 도출 → 기획/마케팅팀과 함께 액션 플랜 논의.

이런 분께 잘 맞아요

수학 천재는 아니어도 숫자·그래프 읽는 건 괜찮음

논리적으로 생각하는 걸 좋아함

“느낌”보다는 “근거”로 말하는 스타일

이 7개를 봇이 한 번에 다 쏟기보다는,

첫 메시지에서 “큰 지도”로 보여주고

사용자가 궁금한 직무를 클릭하면 더 자세한 설명을 펼쳐보는 방식(카드/아코디언)이 좋습니다.

활동 유형:

공모전, 프로젝트, 동아리, 인턴… (이미 페이지 필터에 있음)

예시 의사코드

user_main_track = "전략/경영기획"
user_activity_type = "공모전"   // 또는 null이면 전체

track_to_tags = {
  "전략/경영기획": ["기획/전략", "경영/사업"],
  "마케팅": ["마케팅"],
  "재무/회계": ["재무/회계"],
  ...
}

interest_tags = track_to_tags[user_main_track]

query = {
  활동유형: user_activity_type or 전체,
  관심분야: interest_tags,
  학과: "경영학과" (선택),
  난이도: (학년·경험에 따라 beginner/intermediate)
}
